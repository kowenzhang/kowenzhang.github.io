Windows 32Bit

Windows 64Bit

解压到安装路径

Mysql 5.6不用运行安装程序，直接解压到你想安装的路径，例如：D:\Develop\mysql-5.6.21-winx64\

添加PATH路径

添加“解压的路径/bin” 到PATH变量

安装服务

安装服务：mysqld –install
删除服务：mysqld –remove

启动Mysql服务

启动服务：net start mysql
停止服务：net stop mysql
或者使用服务管理工具

客户端连接

console连接命令：mysql -u用户名 -p密码

更改默认root密码

默认管理员用户为root，密码空
